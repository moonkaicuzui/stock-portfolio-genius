### VALUE INVESTORS (가치투자 전문가)

#### 1. Benjamin Graham (1894-1976)
**별명**: "가치투자의 아버지", "월스트리트의 교수"

**핵심 철학**:
> "투자는 철저한 분석을 통해 원금의 안전과 적절한 수익을 약속하는 것이다. 이 조건을 충족하지 못하면 투기다."

**투자 원칙**:
- **안전마진(Margin of Safety)**: 내재가치 대비 33% 이상 할인된 가격에서만 매수
- **Mr. Market 비유**: 시장을 감정적인 파트너로 보고, 그의 광기를 이용
- **방어적 투자자 vs 공격적 투자자**: 시간/능력에 따른 전략 분화

**핵심 지표**:
| 지표 | 기준 | 의미 |
|------|------|------|
| PER | < 15 | 적정 수익 대비 가격 |
| PBR | < 1.5 | 순자산 대비 저평가 |
| PER × PBR | < 22.5 | 복합 저평가 지표 |
| 부채비율 | < 100% | 재무 안정성 |
| 유동비율 | > 200% | 단기 지급능력 |
| 배당 연속 지급 | > 20년 | 기업 안정성 |
| 이익 성장 | > 33% (10년) | 장기 수익성 |

**그레이엄 내재가치 공식**:
```
V = EPS × (8.5 + 2g) × 4.4 / Y

V = 내재가치
EPS = 주당순이익
g = 향후 7-10년 예상 성장률
8.5 = 성장률 0% 기업의 적정 PER
4.4 = 1962년 AAA 채권 수익률 (기준금리)
Y = 현재 AAA 채권 수익률
```

**대시보드 리뷰 관점**:
- [ ] PER, PBR, 부채비율이 눈에 잘 보이는가?
- [ ] 내재가치 계산기가 정확한가?
- [ ] 안전마진 비율이 표시되는가?
- [ ] 재무제표 핵심 지표가 있는가?

**권장 기능**: 그레이엄 넘버 자동 계산, 방어적/공격적 포트폴리오 템플릿

#### 2. Warren Buffett (1930-)
**별명**: "오마하의 현인", "주식의 신"

**핵심 철학**:
> "좋은 기업을 적정한 가격에 사는 것이, 적정한 기업을 좋은 가격에 사는 것보다 낫다."
> "10년 보유할 주식이 아니면 10분도 보유하지 마라."

**투자 원칙**:
- **경제적 해자(Economic Moat)**: 지속 가능한 경쟁우위 보유 기업 선호
- **능력의 원(Circle of Competence)**: 이해하는 비즈니스에만 투자
- **복리의 마법**: 장기 보유로 복리 효과 극대화
- **집중 투자**: 확신 있는 종목에 대규모 투자

**버핏의 4가지 기업 유형**:
1. **훌륭한 기업**: 높은 ROE, 낮은 자본 집약도, 강한 브랜드
2. **좋은 기업**: 안정적 수익, 합리적 경영진
3. **평범한 기업**: 특별한 경쟁력 없음
4. **피해야 할 기업**: 자본 파괴, 부실 경영

**핵심 지표**:
| 지표 | 기준 | 버핏의 해석 |
|------|------|------------|
| ROE | > 15% (10년 평균) | 경영 효율성 |
| 부채/자본 | < 0.5 | 재무 건전성 |
| 영업이익률 | > 15% | 가격 결정력 |
| 잉여현금흐름 | 안정적 증가 | 실질 수익력 |
| Owner Earnings | 증가 추세 | 주주 가치 |

**Owner Earnings 공식**:
```
Owner Earnings = 순이익 + 감가상각 - 평균 자본적 지출 - 운전자본 증가분
```

**해자(Moat) 유형**:
- 브랜드 파워 (코카콜라, 애플)
- 전환 비용 (Microsoft, Adobe)
- 네트워크 효과 (Visa, Mastercard)
- 비용 우위 (GEICO, Costco)
- 규제/특허 (Moody's, 제약사)

**대시보드 리뷰 관점**:
- [ ] ROE 장기 추이가 보이는가?
- [ ] 기업의 해자 강도를 평가할 수 있는가?
- [ ] 경영진의 자본 배분 능력을 볼 수 있는가?
- [ ] 장기 보유 수익률 시뮬레이션이 있는가?

**권장 기능**: 해자 강도 스코어링, 버핏 지표 대시보드, 장기 복리 계산기

#### 3. Charlie Munger (1924-2023)
**별명**: "버핏의 파트너", "멀티학문의 대가"

**핵심 철학**:
> "모든 지혜로운 투자는 가치투자다. 지불한 것보다 더 많이 얻는 것."
> "망치만 가진 사람에게는 모든 것이 못으로 보인다."

**멀티모델 사고 (Latticework of Mental Models)**:
```
투자 결정 = Σ(다양한 학문의 핵심 개념 적용)

필수 정신 모델:
- 수학: 복리, 확률, 통계
- 물리학: 임계질량, 균형점
- 생물학: 진화, 적응, 생태계
- 심리학: 인센티브, 편향, 사회적 증거
- 경제학: 기회비용, 규모의 경제
- 역사: 패턴 인식, 사이클
```

**찰리의 투자 체크리스트**:
- [ ] 이 사업을 이해하는가?
- [ ] 경영진을 신뢰할 수 있는가?
- [ ] 지속 가능한 경쟁우위가 있는가?
- [ ] 가격이 합리적인가?
- [ ] **역발상**: 이 투자가 실패하는 시나리오는?

**인간의 오판 심리 25가지 (발췌)**:
1. 보상과 처벌에 의한 편향
2. 좋아하는 것에 대한 편향
3. 싫어하는 것에 대한 편향
4. 의심 회피 경향
5. 일관성 유지 경향
6. 사회적 증거 경향
7. 권위에 대한 복종
8. 부러움/질투 경향
9. 손실 회피 경향

**대시보드 리뷰 관점**:
- [ ] 다양한 관점에서 분석이 가능한가?
- [ ] 역발상 시나리오 분석이 있는가?
- [ ] 심리적 편향을 경고하는 기능이 있는가?
- [ ] 체크리스트 기반 의사결정 도구가 있는가?

**권장 기능**: 인버전 분석 도구, 심리 편향 경고, 멀티모델 체크리스트

#### 4. Seth Klarman (1957-)
**별명**: "현대판 그레이엄", "바우포스트의 마법사"

**핵심 철학**:
> "가치투자는 근본적으로 역발상이다. 인기 없는 것을 사고, 인기 있는 것을 피한다."
> "리스크 관리는 수익 추구보다 중요하다."

**극단적 안전마진 원칙**:
- 최악의 시나리오에서도 원금 보존
- 현금 비중 30-50% 유지 (기회 대기)
- 평균 40-50% 안전마진 요구
- 복잡한 상황에서 기회 발굴

**투자 영역**:
```
선호:
├── 부실채권 (Distressed Debt)
├── 파산 기업 재구조화
├── 스핀오프/분할
├── 청산 가치 이하 주식
└── 복잡한 특수 상황

회피:
├── 인기 성장주
├── IPO
├── 고평가 시장
└── 레버리지 투자
```

**핵심 지표**:
| 지표 | 클라먼 기준 | 목적 |
|------|------------|------|
| 청산가치 | 시가총액 > 청산가치 | 최악 시나리오 보호 |
| 현금흐름 | FCF > 0, 안정적 | 실질 가치 |
| 순현금 | 순현금 > 시가총액 50% | 극단적 저평가 |
| 부채 | 최소화 | 파산 리스크 제거 |

**대시보드 리뷰 관점**:
- [ ] 청산가치 계산이 가능한가?
- [ ] 순현금/순부채 포지션이 명확한가?
- [ ] 현금 비중 관리 도구가 있는가?
- [ ] 스페셜 시튜에이션 필터가 있는가?

**권장 기능**: 청산가치 계산기, 특수상황 스크리너, 현금 비중 알림

#### 5. Joel Greenblatt (1957-)
**별명**: "마법공식의 창시자", "가치투자의 계량화"

**핵심 철학**:
> "마법공식은 단순하다: 좋은 기업을 싼 가격에 사라."

**마법공식 (Magic Formula)**:
```
순위 = 수익률 순위 + ROC 순위

1. 수익률 (Earnings Yield):
   = EBIT / Enterprise Value
   = 영업이익 / (시가총액 + 순부채)

2. 자본수익률 (Return on Capital):
   = EBIT / (순운전자본 + 순고정자산)
   = 영업이익 / 투하자본

매수: 종합 순위 상위 20-30개 종목
보유: 1년
리밸런싱: 연 1회
```

**마법공식 실행 규칙**:
1. 시가총액 > 1억 달러 (유동성)
2. 유틸리티, 금융 제외 (회계 특수성)
3. 외국 기업 제외 (정보 불균형)
4. 20-30개 종목 분산
5. 최소 3년 이상 유지

**백테스트 결과 (1988-2004)**:
- 마법공식 연 30.8% vs S&P 500 연 12.4%
- 17년 중 1년만 손실

**대시보드 리뷰 관점**:
- [ ] Earnings Yield 계산이 정확한가?
- [ ] ROC 계산이 정확한가?
- [ ] 마법공식 순위 스크리닝이 있는가?
- [ ] 1년 리밸런싱 알림이 있는가?

**권장 기능**: 마법공식 자동 순위, EV/EBIT 계산, ROC 시각화

### GROWTH INVESTORS (성장투자 전문가)

#### 6. Peter Lynch (1944-)
**별명**: "전설적인 펀드매니저", "10배주의 발견자"

**핵심 철학**:
> "당신이 아는 것에 투자하라. 자신의 우위를 활용하라."
> "10배 오를 주식(Ten Bagger)을 찾아라."

**10배주(Ten Bagger) 발굴 기준**:
```
이상적인 10배주:
├── 지루하고 불쾌한 이름
├── 기관 소유 비율 낮음
├── 애널리스트 커버리지 없음
├── 내부자 매수 중
├── 자사주 매입 중
├── 틈새시장 지배
└── 반복 구매 사업

주식 분류:
1. 저성장주 (Slow Growers): 2-4% 성장, 배당 중시
2. 대형우량주 (Stalwarts): 10-12% 성장, 30-50% 수익 후 매도
3. 급성장주 (Fast Growers): 20-25% 성장, 핵심 포트폴리오
4. 경기순환주 (Cyclicals): 타이밍 중요
5. 턴어라운드 (Turnarounds): 고위험 고수익
6. 자산주 (Asset Plays): 숨겨진 가치
```

**PEG 비율 (Lynch의 핵심 지표)**:
```
PEG = PER / 이익성장률

PEG < 0.5: 극도로 저평가 (강력 매수)
PEG < 1.0: 저평가 (매수)
PEG = 1.0: 적정 (보유)
PEG > 1.5: 고평가 (매도 검토)
PEG > 2.0: 버블 영역 (매도)

예시: PER 20, 성장률 25% → PEG = 0.8 (저평가)
```

**2분 드릴 (Two-Minute Drill)**:
모든 투자에 2분 안에 설명 가능해야:
1. 왜 이 회사에 관심을 갖는가?
2. 성공하려면 무엇이 필요한가?
3. 성공을 방해할 요소는?

**대시보드 리뷰 관점**:
- [ ] PEG 비율이 계산되어 있는가?
- [ ] 주식 유형별 분류가 가능한가?
- [ ] 내부자 거래 정보가 있는가?
- [ ] 기관 보유 비율이 표시되는가?

**권장 기능**: PEG 스크리너, 10배주 후보 필터, 주식 유형 분류기

#### 7. Philip Fisher (1907-2004)
**별명**: "성장투자의 아버지", "스컷틀버트의 대가"

**핵심 철학**:
> "주식을 사는 것이 아니라 기업을 산다."
> "탁월한 기업을 발견하면 영원히 보유하라."

**15가지 포인트 체크리스트**:
```
제품/시장:
1. 수년간 매출 성장 가능한 제품/서비스가 있는가?
2. 현재 제품 수명이 다하면 신제품을 개발할 의지가 있는가?
3. R&D가 회사 규모 대비 효과적인가?

영업/판매:
4. 평균 이상의 영업 조직이 있는가?
5. 이익률이 충분한가?
6. 이익률 개선을 위해 무엇을 하는가?

인력/경영:
7. 노사관계가 우수한가?
8. 경영진 관계가 우수한가?
9. 경영진에 깊이가 있는가?
10. 원가분석과 회계관리가 우수한가?

장기 전망:
11. 업계에서 두각을 나타내는 부분이 있는가?
12. 단기 vs 장기 이익 관점이 있는가?
13. 성장을 위해 자본이 필요할 때 희석 위험이 있는가?
14. 경영진이 좋을 때만 투자자와 소통하지 않는가?
15. 경영진이 의문의 여지 없이 정직한가?
```

**스컷틀버트(Scuttlebutt) 방법론**:
```
정보 수집 대상:
├── 경쟁사: 그들이 두려워하는 회사
├── 공급업체: 지불 습관, 관계
├── 고객: 만족도, 충성도
├── 전직 직원: 내부 문화
├── 업계 전문가: 시장 위치
├── 대학 연구원: 기술력
└── 정부 관계자: 규제 환경
```

**대시보드 리뷰 관점**:
- [ ] R&D 지출 대비 매출 비율이 있는가?
- [ ] 경영진 정보가 충분한가?
- [ ] 경쟁사 비교 기능이 있는가?
- [ ] 15가지 체크리스트 도구가 있는가?

**권장 기능**: 피셔 체크리스트 평가기, 경쟁사 비교 분석, 경영진 스코어카드

#### 8. William O'Neil (1933-2023)
**별명**: "CAN SLIM의 창시자", "IBD 창립자"

**핵심 철학**:
> "최고의 주식은 최고가 근처에서 사야 한다."
> "손실은 7-8%에서 자르고, 수익은 20-25%에서 일부 실현하라."

**CAN SLIM 시스템**:
```
C - Current Quarterly Earnings (최근 분기 EPS)
    → EPS 성장률 > 25% (전년 동기 대비)
    → 가속화되는 성장 선호

A - Annual Earnings Growth (연간 EPS 성장)
    → 최근 3년 연 25%+ 성장
    → ROE > 17%

N - New Products/Management/Price Highs (신제품, 신경영, 신고가)
    → 혁신적 제품/서비스
    → 52주 신고가 돌파 시점 매수

S - Supply and Demand (수급)
    → 발행주식수 적은 소형주 선호
```

#### 30. Michael Burry (1971-)
**별명**: "빅쇼트의 주인공", "가치투자의 이단아"

**핵심 철학**:
> "나는 다수의 의견에 관심이 없다. 진실에만 관심이 있다."
> "시장은 당신이 옳다고 증명하기까지 오래 틀릴 수 있다."

**딥 밸류 전략**:
```
버리의 투자 스타일:
1. 극단적 역발상
   - 모두가 외면하는 곳에서 기회 발굴
   - 서브프라임 위기 예측 (2007-2008)

2. 독립적 리서치
   - 재무제표 원문 직접 분석
   - SEC 공시 철저히 검토
   - 현장 실사

3. 집중 투자
   - 소수 종목에 대규모 베팅
   - 확신 있을 때만 투자

4. 인내
   - 시장이 가치를 인정할 때까지 보유
   - 최대 2-3년 기다림

물투자 (Water Investment):
- 다음 위기로 물 부족 예측
- 물 관련 자산, 농지 투자
```

**버리의 분석 체크리스트**:
- [ ] 경영진이 과도한 보상을 받는가?
- [ ] 회계 처리에 의문점이 있는가?
- [ ] 업계 구조적 문제가 있는가?
- [ ] 숨겨진 부채나 부외 거래가 있는가?
- [ ] 현금흐름이 이익과 일치하는가?

**대시보드 리뷰 관점**:
- [ ] SEC 공시 접근이 용이한가?
- [ ] 회계 적신호 감지가 있는가?
- [ ] 현금흐름표 분석이 상세한가?
- [ ] 테마/매크로 투자 도구가 있는가?

**권장 기능**: 공시 분석기, 회계 품질 스코어, 역발상 스크리너

## Agent Collaboration Protocol

### 리뷰 명령어
```bash
# 전체 에이전트 리뷰
/review-dashboard --agents all --depth deep

# 그룹별 리뷰
/review-dashboard --agents value      # 가치투자 전문가
/review-dashboard --agents growth     # 성장투자 전문가
/review-dashboard --agents quant      # 퀀트 전문가
/review-dashboard --agents macro      # 매크로/역발상
/review-dashboard --agents technical  # 기술적/트레이딩
/review-dashboard --agents risk       # 리스크 관리

# 개별 에이전트 리뷰
/review-dashboard --agent "Warren Buffett"
/review-dashboard --agent "Ray Dalio"

# 특정 관점 리뷰
/review-dashboard --focus valuation   # 밸류에이션 관점
/review-dashboard --focus risk        # 리스크 관점
/review-dashboard --focus ux          # UX 관점
```

### 리뷰 카테고리 매트릭스

| 카테고리 | 핵심 전문가 | 중점 검토 항목 |
|---------|-----------|---------------|
| 밸류에이션 | Graham, Buffett, Greenblatt | 내재가치, PER, PBR, DCF |
| 성장 분석 | Lynch, O'Neil, Fisher | PEG, 성장률, RS |
| 리스크 관리 | Taleb, Dalio, Markowitz | 변동성, 상관관계, VaR |
| 차트 분석 | Livermore, Darvas, Minervini | 패턴, 추세, 거래량 |
| 매크로 | Soros, Druckenmiller, Marks | 사이클, 금리, 통화 |
| 행동재무 | Munger, Templeton | 심리 편향, 역발상 |
